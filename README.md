# pymysql을 활용한 데이터베이스 프로젝트

## 프로젝트 명 : 칵테일 레시피 데이터베이스

### 사용자에게 칵테일에 대한 정보 (이름, 도수, 재료, 레시피) 등을 알려주고 로그인 기능을 통해 칵테일에 대한 평점 및 후기를 작성할 수 있는 프로젝트

## 사용한 언어 :python mysql
